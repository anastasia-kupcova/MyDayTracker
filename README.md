# MyDayTracker (Дневник самочувствия)

Профессиональное desktop-приложение для отслеживания психоэмоционального и физического состояния с визуализацией данных в виде календаря с цветовой индикацией, графиками и рекомендациями. Создано на Java с современным графическим интерфейсом. Позволяет пользователям ежедневно отмечать ключевые показатели самочувствия, анализировать динамику через графики и диаграммы, а также записывать важные события месяца.

## О проекте

### Почему был создан этот продукт?

MyDayTracker появился как практическая реализация цифровой системы отслеживания психоэмоционального и физического состояния в формате локального desktop-приложения. Проект объединяет потребности пользователей в ежедневном мониторинге самочувствия с инструментами визуализации данных в виде цветного календаря и аналитической статистики. Разработка велась как комплексный пример полноценного приложения с архитектурой MVC, реляционной базой данных и многооконным интерфейсом.

### Какую проблему он решает?

Приложение предоставляет единую цифровую среду для отслеживания состояния без зависимости от облачных сервисов. Оно автоматизирует процесс ведения дневника самочувствия, заменяя разрозненные заметки и бумажные планировщики структурированной системой с визуальной обратной связью в виде цветового календаря и аналитических графиков.

### В чём его ключевые преимущества?

**Полная локальность и безопасность:** Все данные хранятся в локальной базе MySQL на вашем компьютере  
**8 трекеров в одном приложении:** Рейтинг дня, энергия, социальный контакт, тревожность, погода, сон, сновидения и событие месяца  
**Цветовая визуализация данных:** Интуитивный календарь с цветовой индикацией (heatmap) для наглядного анализа состояния за год  
**Комплексная статистика:** Линейные графики, круговые и столбчатые диаграммы по всем трекерам  
**Гостевой режим:** Просмотр данных без регистрации (без возможности сохранения)  
**Готовое рабочее решение:** Полнофункциональное приложение с типовыми сценариями отслеживания состояния «из коробки»

## Технологии

Для работы приложения требуется:

**Java Runtime Environment (JRE) 17 или выше:** [Скачать с официального сайта](https://www.oracle.com/java/technologies/downloads/)  
**СУБД MySQL 8.0 или выше:** [Скачать установщик](https://dev.mysql.com/downloads/mysql/)

### Архитектура проекта:

| Компонент | Технология |
|---|---|
| Язык и интерфейс | Java 17, JavaFX для современного GUI |
| База данных | MySQL 8.0 с полной схемой пользователей, трекеров, событий месяца и настроек |
| Архитектура | MVC (Model-View-Controller) с чётким разделением ответственности |
| Дополнительные компоненты | DAO-паттерн для доступа к данным, шифрование паролей (BCrypt), защита от SQL-инъекций (PreparedStatement) |

## Быстрая установка и запуск

Вы можете запустить приложение за несколько минут, используя готовые файлы:

### 1. Установите необходимое ПО

Установите Java (JRE) по ссылке выше.  
Установите MySQL Server. Во время установки запишите заданный вами пароль для пользователя root.

### 2. Настройте базу данных

Скачайте готовый дамп базы данных.  
Откройте командную строку MySQL (или общую командную строку/терминал).  
Выполните команду для импорта данных, указав правильный путь к скачанному файлу. Вас попросят ввести пароль пользователя root MySQL.

### 3. Запустите приложение

Скачайте приложение из репозитория.  
Запустите его двойным кликом или из командной строки.

**Важно:** Перед запуском убедитесь, что сервис MySQL Server запущен в вашей системе.

## Возможности

MyDayTracker предоставляет полный цикл отслеживания психоэмоционального и физического состояния:

**Система пользователей:** Регистрация и авторизация с хешированием паролей (BCrypt), гостевой режим без сохранения данных  
**Отслеживание рейтинга дня:** Оценка общего самочувствия по шкале 0–5 с цветовой индикацией от серого до зеленого  
**Отслеживание энергии:** Уровень энергии утром и вечером по шкале 0–100% с градиентной цветовой индикацией от красного до зеленого  
**Отслеживание социального контакта:** Оценка социальной активности по шкале 0–5 с цветовой индикацией от черного до ярко-желтого  
**Отслеживание тревожности:** Уровень тревоги 0–3 (не было, низкий, средний, высокий) с цветовой индикацией от зеленого до красного  
**Отслеживание погоды:** Выбор из 8 типов погоды (солнечно, облачно, пасмурно, дождь, дождь с грозой, облачно и снег, снег, ветер) с цветовой индикацией  
**Отслеживание сна:** Количество часов сна по шкале 1–6 (≤4 ч, 5 ч, 6 ч, 7 ч, 8 ч, 9+ ч) с цветовой индикацией от темно-красного до темно-зеленого  
**Отслеживание сновидений:** Выбор из 9 типов снов (не было, не запомнился, смешной, радостный, грустный, странный, абсурдный, страшный, кошмар) с цветовой индикацией  
**Событие месяца:** Текстовое описание главного события за месяц с визуальной индикацией на карточке месяца  
**Календарь:** 12×31 таблица с цветовыми ячейками, буквенные обозначения месяцев, неактивные ячейки для месяцев с <31 днями  
**Переключение годов:** Выбор года из выпадающего списка с автоматической перезагрузкой всех данных  
**Легенда:** Справочник цветов и значений для каждого трекера  
**Статистика:** Еженедельный, ежемесячный и ежегодный просмотр с линейными графиками, круговыми и столбчатыми диаграммами  
**Линейные графики:** Рейтинг дня по дням, энергия утром и вечером на одном графике  
**Круговые диаграммы:** Распределение уровней тревожности и типов погоды  
**Столбчатые диаграммы:** Распределение часов сна, уровней социального контакта и типов сновидений

## Использование

### Начало работы

При первом запуске зарегистрируйтесь, введите логин и пароль, или войдите под существующими данными. Также доступен гостевой режим для ознакомления.

### Главный экран

После входа откроется рабочее пространство с вкладками для каждого трекера, календарем, статистикой и событиями месяца.

### Основные функции (панель вкладок)

| Раздел | Описание |
|---|---|
| **Рейтинг дня** | Календарь с цветовой индикацией оценки самочувствия 0–5 |
| **Энергия** | Две таблицы (утро/вечер) с градиентной цветовой индикацией 0–100% |
| **Социальный контакт** | Календарь с цветовой индикацией уровня социальной активности 0–5 |
| **Тревожность** | Календарь с цветовой индикацией уровня тревоги 0–3 |
| **Погода** | Календарь с цветовой индикацией типа погоды 1–8 |
| **Сон** | Календарь с цветовой индикацией часов сна 1–6 |
| **Сновидения** | Календарь с цветовой индикацией типа снов 1–9 |
| **События месяца** | Карточки месяцев 4×3 с текстовым описанием событий |
| **Статистика** | Графики и диаграммы с выбором периода (неделя/месяц/год) |

### Работа с данными

Клик по ячейке календаря открывает диалог выбора значения для трекера  
Клик по карточке месяца открывает диалог ввода текстового события  
Все изменения сохраняются автоматически через транзакции БД  
Переключение года обновляет все календари и статистику

## Разработка и тестирование

Проект построен на архитектуре MVC с чётким разделением:

**Модели:** User, MonthEvent, UserSettings  
**Контроллеры:** LoginController, MainController для каждого окна  
**Представления:** FXML-файлы с современным UI и CSS-стилизацией  
**DAO-слой:** UserDAO, DataDAO, MonthEventDAO для работы с каждой сущностью в БД

### Тестирование

Проведено комплексное тестирование, охватывающее основные сценарии использования:

| Тест-кейс | Сценарий | Охватываемый функционал |
|---|---|---|
| **Т1-Т9** | Полная регистрация и вход с переходом к каждому трекеру | Аутентификация, регистрация, навигация по всем трекерам |
| **Т10-Т18** | Гостевой режим и просмотр всех трекеров | Гостевой вход, просмотр данных, блокировка сохранения |
| **Т19-Т27** | Авторизация и работа с каждым трекером | Авторизация, сохранение данных, цветовая индикация |
| **Т28-Т33** | Переходы между трекерами и статистикой | Навигация между модулями, работа со статистикой |

Интеграционное тестирование проводилось на основе графа переходов между окнами приложения, обеспечивая покрытие всех ключевых пользовательских сценариев.
